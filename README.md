# 💳 Financial Fraud Analysis Dashboard - Tableau

An interactive Tableau dashboard developed as a **Capstone Project** to analyze financial transaction data, identify fraudulent activities, monitor transaction trends, and uncover business insights through visual analytics.

---

# 📌 Project Overview

Financial fraud has become one of the biggest challenges for banking and financial institutions. This dashboard helps analyze transaction behavior, identify suspicious patterns, monitor transaction amounts, and visualize fraud distribution across different locations and customer segments.

The dashboard enables business users to quickly identify fraud-prone areas and understand transaction trends for informed decision-making.

---

# 🎯 Business Objectives

- Analyze fraudulent transactions geographically
- Monitor transaction amount trends over time
- Compare spending patterns across customer genders
- Identify high-risk transaction locations
- Track inflation-adjusted transaction amounts
- Discover category-wise transaction patterns

# 📊 Dashboard Features

## 🌍 Fraud Location Analysis
- Visualizes fraudulent transactions across geographical locations
- Highlights fraud hotspots using map visualization

## 📈 Inflation Adjusted Transaction Analysis
- Tracks inflation-adjusted transaction amounts over time
- Helps identify unusual spikes in financial activity

## 💰 Monthly Transaction Trends
- Shows month-wise transaction amount patterns
- Detects seasonal fluctuations and anomalies

## 👨‍👩‍👧 Gender-wise Spending Analysis
- Compares spending behavior between male and female customers
- Identifies category-wise spending distributions

## 🗺 Geographic Transaction Distribution
- Displays transaction locations using latitude and longitude
- Enables regional transaction analysis

# 📈 Key Insights

- Fraudulent transactions are concentrated in specific geographic regions.
- Monthly transaction volumes fluctuate throughout the year, indicating seasonal spending behavior.
- Inflation-adjusted transaction values reveal unusual peaks that may require further investigation.
- Spending behavior varies across customer genders.
- Interactive filters enable dynamic exploration of fraud patterns and transaction categories.
